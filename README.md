# 🏠 Miami Housing ML Analysis

## 📌 Overview
This project analyzes housing data from Miami to predict property prices and classify high-value homes using machine learning models.

The project combines regression and classification techniques to extract insights and evaluate model performance.

---

## 🎯 Objectives
- Predict housing prices using regression models
- Identify whether a property is above average price
- Compare multiple machine learning algorithms
- Evaluate model performance using metrics and visualizations

---

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Statsmodels
- Matplotlib & Seaborn

---

## 📊 Project Workflow

### 1. Data Preprocessing
- Loaded dataset and renamed target column
- Removed unnecessary columns
- Split data into training and testing sets

### 2. Regression Models
Implemented and compared:
- Decision Tree Regressor
- Random Forest Regressor
- K-Nearest Neighbors Regressor

Evaluation Metrics:
- RMSE (Root Mean Squared Error)
- R² Score

---

### 3. Model Comparison
- Trained multiple models
- Selected the best model based on R² score

---

### 4. Visualization
- Actual vs Predicted values plots
- Scatter plots for model performance comparison

---

### 5. Classification Task
Created a classification problem:
- Classified houses as **above average price** or not

Models used:
- K-Nearest Neighbors
- Random Forest Classifier
- Logistic Regression

Evaluation Metrics:
- Precision
- Recall
- F1 Score
- Confusion Matrix

---

## 📈 Key Insights
- Random Forest performed best in predicting housing prices
- Classification models effectively identified high-value properties
- Visualization helped compare model accuracy and errors
